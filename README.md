# WayFinder Platform

> Connecting families with trusted support.

WayFinder is a platform that helps families discover trusted therapists, centres, and specialists through community recommendations.

The platform is designed to make finding support simple, transparent, and accessible.

---

## ✨ Vision

Finding the right support shouldn't be difficult.

WayFinder brings together trusted recommendations from families to help others discover providers with confidence.

---

## 🚀 MVP Features

- 🔍 Search providers
- 📍 Browse by locality
- 🧩 Browse by service
- ⭐ Community trust badges
- 💬 WhatsApp contact
- 🗺️ Google Maps integration
