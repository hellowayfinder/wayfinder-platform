# Local Development

## Clone

```bash
git clone https://github.com/hellowayfinder/wayfinder-platform.git
```

---

## Install

```bash
npm install
```

---

## Run

```bash
npm run dev
```

---

## Build

```bash
npm run build
```

---

## Branches

- main
- develop

Develop on `develop`.

Merge stable work into `main`.
