# MVP Scope

This document defines the exact scope of Version 1.

Anything not listed here belongs in a future release.

---

## Provider Directory

- Provider Listing
- Provider Details
- Search
- Filter by Service
- Filter by Locality

---

## Parent Features

- Submit Feedback

## Analytics

- Google Analytics

---

## Deployment

- Production Deployment

## Community

- Ratings

## AI

## Expansion

- Multiple Cities
- Multiple Languages

---

## Mobile

- Android App
- iOS App

---

# MVP Definition

The MVP is complete when a parent can:

1. Open WayFinder
2. Search for providers
3. Filter providers
4. View provider information
5. Recommend a provider
6. Submit feedback

without requiring any manual assistance.

---

# Guiding Rule

If a feature does not directly help parents discover trusted providers, it should not delay the MVP.
