# Contributing

## Branch Strategy

main
- Stable
- Production ready

develop
- Daily development

---

## Before every commit

Run

```bash
npm run build
```

If the build fails

Do not commit.

---

## Commit Messages

Use Conventional Commits.

Examples

- feat:
- fix:
- docs:
- refactor:
- style:
- chore:

---

## Pull Requests

Future contributors should work from feature branches into develop.
